# Features

Estructura modular por dominio de negocio para SPORTMATCH Mobile.

- `auth`: autenticacion, registro y onboarding.
- `home`: home del usuario.
- `explore`: exploracion de oportunidades/jugadores/equipos.
- `create`: wizard de creacion de partidos.
- `matches`: hub de partidos y detalle.
- `chat`: conversaciones por partido.
- `teams`: gestion de equipos e invitaciones.
- `profile`: perfil y ajustes del usuario.
- `admin`: herramientas de administrador.
- `notifications`: centro de notificaciones y deep links.
